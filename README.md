# Interactive Calendar Component

## 🚀 Features
- Wall calendar UI design
- Date range selection (start → end)
- Notes section (monthly + range-based)
- Responsive layout (desktop + mobile)
- Weekend highlighting
- Previous/next month faded dates

## 🛠 Tech Used
- React (Vite)
- JavaScript
- CSS (inline styling)
